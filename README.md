# Throne8 Entry Client

A modern web application built with Next.js for user authentication, social networking, and professional networking features.

## ✅ Completed Features

### Authentication
- ✅ Multi-step signup (4-5 steps)
- ✅ Login with email and password
- ✅ Google OAuth integration
- ✅ GitHub OAuth integration
- ✅ Forgot password with OTP
- ✅ Password reset functionality
- ✅ Resend password request OTP

### Dashboard & Posts
- ✅ Dashboard redirect after authentication
- ✅ Display all posts from all users
- ✅ Post mood-based thoughts feature
- ✅ Post action buttons with full functionality

### User Features
- ✅ Profile management (frontend implementation complete)
- ✅ Connections and networking
- ✅ Analytics integration (coming soon)

## 🚀 Tech Stack

- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: OAuth 2.0 (Google, GitHub)

## 📦 Installation

```bash
npm install
```

## 🏃 Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
